**Program Details**

The MotorPH Payroll System is a Java program that computes employee payroll. The system reads employee data from a file and calculates the salary based on the total hours worked. It allows payroll staff to process payroll for one employee or all employees in the system.

The program calculates the gross salary and applies government deductions such as SSS, PhilHealth, Pag-IBIG, and tax, which are deducted during the second cutoff. After all deductions are applied, the system displays the net salary together with employee information such as employee number, name, birthday, hours worked, gross salary, and deductions.
